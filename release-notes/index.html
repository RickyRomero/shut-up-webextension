Shut Up 3.0

Released April 17th, 2017

You can finally hide Shut Up's toolbar icon without giving up any control. This update also contains several important security fixes.

  • Rewrote the extension from scratch.
  • Added a customizable keyboard shortcut.
  • Added a context menu.
  • Added a way to force an update for shutup.css.
  • Translated the extension to several new languages. Contributions are welcome!
    • Traditional and Simplified Chinese
    • Spanish
    • Arabic
  • Dramatically improved reliability when hiding and showing comments.
  • Fixed an issue where the icon could incorrectly become disabled on Facebook, SoundCloud, and other sites which use the history.pushState() API.
  • Increased the security of the stylesheet updater and whitelist.
  • Fixed how the toolbar icon shows up in incognito tabs.
  • Refined the design of the Options screen.
  • Fixed typose in release notes.
