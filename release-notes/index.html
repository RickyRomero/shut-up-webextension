Shut Up 8.1

Released xx xxxx, 2024

This release ensures compatibility with future browser releases and further strengthens security.

Due to new browser requirements, Shut Up no longer includes a stylesheet update component. Your browser is now responsible for these updates, automatically installing them when updating the extension.

  • Restored the changes from the 8.0 update, following closer scrutiny and validation.
  • Added an operation queue to address bugs caused by race conditions.
  • Optimized various image files. Thanks to Frazer Smith for the contribution.

This update changes how comments are blocked across many sites. Please report any issues anonymously using this form. Thank you!

Shut Up 8.0

Released September 21st, 2022

This release ensures compatibility with future browser releases and further strengthens security.

Due to new browser requirements, Shut Up no longer includes a stylesheet update component. Your browser is now responsible for these updates, automatically installing them when updating the extension.

  • Upgraded to Manifest v3 for future browser compatibility.
  • Switched to new native WebExtensions APIs for adding blocking rules to pages.
  • Updated the Options page to support narrow window widths.
  • Fixed an issue in the Options page that prevented links from opening in a full tab.
  • Removed the ability to force stylesheet updates in browsers based on Chrome.

This update changes how comments are blocked across many sites. Please report any issues anonymously using this form. Thank you!

Shut Up 7.0

Released October 12th, 2020

Thank you for all of your feedback on the previous release.

I've reverted the changes from Shut Up 6. It improved comment blocking by using more general CSS selectors. However, it also blocked a lot of things that aren't comments. This broke many sites in lots of unexpected ways.

Future releases will continue to use the old, targeted blocking approach. This will miss comments sometimes, so please continue to report any issues anonymously using this form.

Shut Up 5.0

Released December 21st, 2019

Shut Up is now available on Microsoft Edge 76 and newer, and unofficially on Opera 65 and newer through the Chrome Web Store.

  • Fixed comment blocking on Firefox 72 and newer.
  • Resolved various race conditions using improved initialization logic. This fixed an issue where the context menu item wouldn't install correctly.
  • Strengthened allowlist encryption using the Web Crypto API.
  • Improved stylesheet update logic to handle (some) Unicode characters.
  • Strengthened Content Security Policy to prepare for upcoming Manifest v3 changes.

An all-new version of Shut Up for Safari 12 and 13 on macOS is in the works. Follow me for updates!

Shut Up 3.0

Released April 17th, 2017

You can finally hide Shut Up's toolbar icon without giving up any control. This update also contains several important security fixes.

  • Rewrote the extension from scratch.
  • Added a customizable keyboard shortcut.
  • Added a context menu.
  • Added a way to force an update for shutup.css.
  • Translated the extension to several new languages. Contributions are welcome!
    • Traditional and Simplified Chinese
    • Spanish
    • Arabic

  • Increased the security of the stylesheet updater and allowlist.
  • Fixed how the toolbar icon shows up in incognito tabs.
  • Refined the design of the Options screen.
  • Fixed typose in release notes.
