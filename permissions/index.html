Welcome to Shut Up

Sublime comment blocking for {browser}.

Get Started

To block comment sections, Shut Up needs access to the webpages you visit. It only uses this access to hide comment sections.

Learn More

Shut Up inserts CSS rules into every webpage you visit to block comment sections before you can see them. This function requires permission to change the websites you visit.

Shut Up doesn't read the contents of the pages you visit, nor does it contact any other services, so it can't track your browsing activity.

I'm committed to your security and privacy. Read my plain-English privacy policy here. Shut Up is an open-source project, so anyone can audit its behavior.
