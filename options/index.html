This extension adds CSS rules into every website you visit to block comment sections before you can see them. This function requires permission to change the websites you visit.

Shut Up won't read the contents of the sites you visit, nor will it contact any other services. It can't track you.

I'm committed to your security and privacy. Read my plain-English privacy policy here. Shut Up is an open-source hobby project, so anyone can audit its behavior.
